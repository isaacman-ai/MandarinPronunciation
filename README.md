# Mandarin Pronunciation Learning App

A simple web application to learn Mandarin pronunciation. Click on the cards to hear the pronunciation of words.

## Features
- **Visual Vocabulary**: Words are presented with Chinese characters and Pinyin.
- **Text-to-Speech**: Click on any card to hear the native pronunciation using the Web Speech API.
- **Responsive Design**: Works on desktop and mobile.

## How to Run Locally
1. Simply open the `index.html` file in your web browser.
2. Click on the cards to test the audio.

## How to Deploy (Make it Public)
You can easily host this for free using **GitHub Pages**.

### Method 1: Uploading to GitHub
1. Create a new repository on [GitHub](https://github.com/new).
2. Upload the files (`index.html`, `style.css`, `script.js`, `data.js`) to the repository.
3. Go to the repository **Settings**.
4. Click on **Pages** in the left sidebar.
5. Under **Source**, select `main` (or `master`) branch.
6. Click **Save**.
7. Wait a few moments, and GitHub will provide you with a public link (e.g., `https://username.github.io/repo-name/`).

### Method 2: Using Command Line (if you have Git installed)
```bash
# Initialize a new git repository
git init

# Add all files
git add .

# Commit changes
git commit -m "Initial commit"

# Link to your new GitHub repository (replace URL)
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git

# Push to GitHub
git push -u origin main
```
Then follow steps 3-7 from Method 1.
